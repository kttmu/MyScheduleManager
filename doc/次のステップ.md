## ⚠️ よくあるトラブルと対処法

### トラブル 1: モジュールが見つからない
**エラー**:
```
ModuleNotFoundError: No module named 'work_scheduler'
```

**原因**: Pythonのパスが正しく設定されていない

**対処法**:
```bash
# プロジェクトルートディレクトリで実行
cd C:\path\to\WorkScheduler
python -m work_scheduler.main sample.xlsx
```

### トラブル 2: openpyxl が見つからない
**エラー**:
```
ModuleNotFoundError: No module named 'openpyxl'
```

**原因**: パッケージがインストールされていない

**対処法**:
```bash
pip install -r requirements.txt
```

### トラブル 3: Excel列が見つからない
**エラー**:
```
ValueError: 必須列が見つかりません: ['業務テーマ', '成果物', 'タスク']
```

**原因**: Excelファイルの列名が異なる

**対処法**:
1. Excelファイルを開く
2. 1行目に以下の列名を設定:
   - A列: 業務テーマ
   - B列: 成果物
   - C列: タスク
   - D列: 開始時期
   - E列: 終了時期
   - F列: ステータス

### トラブル 4: 日付フォーマットエラー
**エラー**:
```
ValueError: 日付形式が無効です
```

**原因**: 日付が非対応の形式

**対処法**: 日付を以下の形式に統一
- ✅ 2026-01-15 (YYYY-MM-DD) - 推奨
- ✅ 2026/01/15 (YYYY/MM/DD)
- ✅ 2026年01月15日
- ❌ 1/15/2026 (MM/DD/YYYY) - 非対応
- ❌ 15-01-2026 (DD-MM-YYYY) - 非対応
