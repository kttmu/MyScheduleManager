# 指示
以下の要件を満たすpythonのスクリプトの基本設計を行うこと

## 1. ツールの目的
- 業務計画の記載されたエクセルファイルを読み込み、進捗状況をリッチなグラフィカルなユーザーインターフェースを備えたhtmlファイルを作成する
-  予定進捗と現状の進捗をグラフと数値で比較できるようにする
-  遅れたタスク/成果物/テーマをリスト化して表示できるようにする
-  進捗中のタスク/成果物/テーマをリスト化して表示できるようにする
-  業務テーマ一覧の進捗状況をテーマごとにグラフ化できるようにする
-  後は良しなに
-  

## 2. インプット情報
エクセルファイル

## エクセル列要素
業務テーマ
成果物
タスク
開始時期
終了時期

ただし、エクセルファイルはWBS形式のデータ構造となっているため、各テーマ、成果物は最初の行にしか書かれていないこととする。
