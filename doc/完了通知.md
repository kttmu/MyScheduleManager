## 🎯 主要機能

### 1. Excel自動読み込み
- WBS形式のExcelを自動認識
- 6つの必須列を検証
- エラーハンドリング完備

### 2. 階層構造の自動構築
```
Theme（ビジネステーマ）
└─ Deliverable（成果物）
   └─ Task（タスク）
```

### 3. 期間加重平均による進捗計算
```
進捗率 = Σ(タスク進捗 × タスク期間) / Σ(タスク期間)
```
- 期間の長いタスクの進捗を優先的に反映
- より実測に近い進捗率の算出

### 4. スマートなステータス判定
- **完了**: 100%進捗
- **着手中**: 50%進捗
- **未着手**: 0%進捗
- **On Track**: 実績 ≥ 計画進捗
- **At Risk**: 計画-10% ≤ 実績 < 計画
- **Delayed**: 実績 < 計画-10% または 期限超過

### 5. Jiraダッシュボード風UI
- 5つの可視化セクション
- レスポンシブデザイン
- カラーコーディング（Green/Yellow/Red）
- 埋め込みCSS（外部ファイル不要）
